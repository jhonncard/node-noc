# Proyecto NOC

El objetivo es crear una serie de tareas usando Arquitectura Limpia con TypeScript.

## DEV

1. clonar el archivo env.template a .env.**
2. configurar las variables de entorno.**

    ```text
        PORT=

        MAILER_EMAIL =
        MAILER_SERCRET_KEY =
        PROD = FLASE
        DEV = TRUE 
        QA = FALSEs
    ```

3. Ejecutart el comando

    ```bash
    npm install
    ```

4. Ejecutar

    ```bash
    npm run dev
    ```
